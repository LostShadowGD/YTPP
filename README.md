# YTPP
A Firefox Add-On which tries to fix the YT desktop web UI.

## Features (that are supposed to work)
- Removes AI
- Reduces size of thumbnails on the right of the video player
- Hides Playables
- Return capital letters in abbreviations (```k``` -> ```K```, etc)
- Return the old video tile metadata (from ```▶ 1k  2 yr ago``` to ```1k views • 2 years ago```)
If one of these features does not work, use the Issues tab to create a bug report.

## Planned Features
- UI for toggling features (along with MAJOR speed and efficiency boosts to the code in general)
- Option to remove unique banners/logo for events
- Option to remove news section
- Option to remove shorts

## Installation
Click the ```Releases``` tab to the left, and click the latest .xpi file to download it. Then, in Firefox, go to ```about:addons```, click the cog next to ```Manage Your Extensions```, select ```Install Add-On From File``` and open the .xpi file you downloaded in the resulting window.
